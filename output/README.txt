Скрипт для чистки кеша 1с

Скрипт работает под правами пользователя, под которым запущен.
Перед работой скрипта стоит закрыть запущенную 1С у пользователя, для которого чистится кеш.

Параметры запуска
-force		:	применение скрипта без отображения диалоговых окон
-safe		:	дополнительно создается резервная копия папки кеша рядом с основной папко	
			- переименование в формат папка_ДД_ММ_ГГГГ
-all_usr	:	чистка для всех пользователей системы (понадобятся права администратора)
-log		:	при выполнении скрипта создается лог-файл
